## 🛠️ Features!
- ⭐️ No internet required.
- ⭐️ Play audio instead of reading.
- ⭐️ Adaptive themes + Material-You.
- ⭐️ Simple and easy to use.
- ⭐️ Fast and lightweight.
- ⭐️ Free Open source & secure.
